# CLAUDE.md

## Commit Convention

Use the following commit message format:

- `post: <book title>` — adding a new book note post
- `post: <title1>, <title2>` — adding multiple posts
- `update: <book title>` — modifying an existing post
- `fix: <brief description>` — fixing formatting, front matter, or bugs
- `config: <brief description>` — changing site configuration (_config.yml, etc.)
